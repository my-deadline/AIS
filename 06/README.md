>Горький - фильтр кофе, лунго, кофе по турецки

**5) Поболтать о кофе**

Узнать страну производства конкретного кофе, степень обжарки, насколько ароматный, насыщенный, кислый, тельный вкус

>Если более 50% - ответ да, достаточно
>Менее 50% - не совсем
